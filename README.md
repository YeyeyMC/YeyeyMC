### PokeWalk

PokeWalk is a mobile application built with React Native and Expo that tracks the user's steps and unlocks Pokémon in a personal Pokédex based on walking progress.

**Key features:**
- Firebase Authentication
- Firestore database integration
- REST API consumption
- Native gyroscope integration
- Responsive and reusable UI components

**Technologies:** `React Native` `TypeScript` `Firebase` `REST APIs`

[View Repository](https://github.com/YeyeyMC/PokeWalk) - [View In YouTube](https://www.youtube.com/shorts/15coSGb2KGc) - [View In Expo](https://snack.expo.dev/@yeisonmunoz/pg29_yeisonmunoz_mobiledev_pokewalk_a3)

### 🌐 Flappy Bird Telemetry System

Full-stack telemetry system that collects and visualizes gameplay data from a Unity WebGL Flappy Bird Game. The project integrates Unity, Firebase, React, and Python to track player behavior,
generate analytics, and display real-time graphs and leaderboards through a web interface with admin and player roles.

**Key features:**
- Used Firebase for data storage and authentication
- Implemented real-time data tracking, visualization, and leaderboard systems
- Integrated Unity WebGL with a real-time React dashboard 

**Technologies:** `React` `JavaScript` `Python` `Firebase` `Unity`

[View Repository](https://github.com/YeyeyMC/Pipelines_GameWebApp) - [Live Demo](https://yeyeymc.github.io/Pipelines_GameWebApp/)
